# vuepress

VuePress基于node 8.0+
1.全局安装 vuepress：npm install -g vuepress
2.初始化项目：npm init
3.创建文件夹docs
4.在 package.json 文件里添加两个启动命令

"scripts":{ 
     "dev": "vuepress dev docs",  
     "build": "vuepress build docs"
 }

5.运行：npm run dev

目录结构
vuepressBlog
├─── docs
│	|___ README.md
|   
└── package.json

6.运行: npm run build

目录结构
vuepressBlog
├─── docs
│   ├── README.md
│   └── .vuepress
│       ├── dist
│ 		├── public
|		├── styles
|		|	|__palette.styl  创建修改原样式
|		|
│       └── config.js
└── package.json

7.在.vuepress文件中创建config.js配置文件

module.exports = {
	title: 'WM_Blog', //设置网站标题
	description : 'WM_Blog',
	base : '/',
	themeConfig : {
	nav : [
		{ text: '描述', 
		  link: '路径' 
		}
	],
	sidebar: {
		'/' : [
		"/", //根目录README.md内容
		]
	},
	sidebarDepth : 2
	}
    }

8.将个人博客部署至github

创建文件deploy.sh
内容：

#!/usr/bin/env sh
# 确保脚本抛出遇到的错误
set -e
# 生成静态文件
npm run build

# 进入生成的文件夹
cd docs/.vuepress/dist

# 如果是发布到自定义域名
echo 'www.wumei.blog' 

git init
git add -A
git commit -m 'deploy'

# 如果你想要部署到 https://<USERNAME>.github.io
git push -f git@github.com:Wumei222/Wumei222.github.io.git master

# 如果发布到 https://<USERNAME>.github.io/<REPO>  REPO=github上的项目
# git push -f git@github.com:<USERNAME>/vuepress.git master:gh-pages

cd -

9.在package.json的scrips中添加 "deploy": "bash deploy.sh"
10.如需修改重新上传github打开git bash命令窗输入sh deploy.sh
